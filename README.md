# College Management System (Python)

Simple command-line College Management System written in Python.

Contents
- `main.py` — application entry point
- `Student.py`, `faculty.py`, `Course.py`, `Fees.py` — domain modules
- `data/` — JSON data files (courses.json, faculty.json, fees.json, students.json)

Quick start

1. Install Python 3.8+ (recommended)
2. Run the application:

```bash
python3 main.py
```

Notes
- This repo stores data in `data/` as JSON files. Back up those files before large edits.
- Add a virtual environment and `requirements.txt` if external packages are introduced.
